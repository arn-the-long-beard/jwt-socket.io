A small library for authentication with a user, JWT and socket.io

## Installation

  `npm install jwt-socket.io`

## Usage

  ```javascript
  const io = require('socket.io')(server)
  const ioAuthCheck = require('io-auth-check')
  io.use(ioAuthCheck)
  ```
